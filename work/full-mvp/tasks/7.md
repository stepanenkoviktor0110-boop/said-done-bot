# Task 7: Rating + feedback flow (comment → consent → save)

## Description
Implement post-task-extraction feedback collection.

## What to do

In `entrypoints/telegram_bot.py`:
1. Callback handler `@router.callback_query(F.data.startswith("rate:"))`:
   - If not awaiting_feedback → FEEDBACK_EXPIRED
   - rating=5: save_feedback(vr_id, 5, None, 0), clear state, FEEDBACK_THANKS
   - rating<5: set awaiting_comment=True, ask FEEDBACK_ASK_COMMENT

2. Text handler for comment:
   - If awaiting_comment: save comment, set awaiting_consent=True
   - Show buttons: [Да] [Нет] for voice consent

3. Callback handler `@router.callback_query(F.data.startswith("consent:"))`:
   - consent=yes: download audio from Telegram, save to `data/voices/{vr_id}.oga`, update VR audio_path
   - consent=no: just save feedback
   - save_feedback(vr_id, rating, comment, consent)
   - Clear state, FEEDBACK_SAVED

4. Decision 8: if new voice arrives during pending feedback → silently abandon (in handle_voice)
