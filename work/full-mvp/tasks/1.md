# Task 1: Database layer — schema, migration, query functions

## Description
Create SQLite database layer with schema, init, and all query functions.

## What to do

1. Create `core/db.py`:
   - `init_db(db_path)` — connect, set WAL + foreign_keys, create tables
   - `get_db()` — return connection singleton
   - Schema: users, voice_requests, feedback, survey_responses (from tech-spec)

2. Create `core/db_ops.py` — all query functions:
   - `upsert_user(telegram_user_id, telegram_username)` → row
   - `get_user(telegram_user_id)` → row | None
   - `create_voice_request(user_id, file_id, duration)` → row
   - `update_voice_request(vr_id, task_count, transcript_length, action_type, summary_length, audio_path)`
   - `get_voice_request(vr_id)` → row | None
   - `save_feedback(vr_id, rating, comment, voice_consent)` → row
   - `decrement_trial(user_id)` → new trial_remaining
   - `complete_survey(user_id)` — phase=2, trial_remaining=10
   - `block_survey(user_id)` — survey_blocked=1
   - `advance_survey(user_id)` — survey_progress += 1
   - `save_survey_response(user_id, question_num, answer, is_adequate, rejection_reason)`

3. Update `entrypoints/telegram_bot.py`:
   - Add `init_db(cfg["db"]["path"])` in main()
   - Import db_ops functions

## Verification
```bash
python3 -c "
from core.db import init_db; init_db(':memory:'); print('DB init OK')
from core.db_ops import upsert_user; u = upsert_user(123, 'test'); print(f'User OK, trial={u[\"trial_remaining\"]}')
assert u['trial_remaining'] == 20, f'Expected 20, got {u[\"trial_remaining\"]}'
"
```
