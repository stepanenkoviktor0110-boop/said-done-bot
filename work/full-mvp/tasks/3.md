# Task 3: Task extraction service — prompt file + parser + LLM wrapper

## Description
Create task extraction module that reads system prompt from file, calls LLM, and parses response.

## What to do

1. Create `prompts/task-extraction.md` — system prompt (copy from МаркетБот `prompts/task-extraction.md` with adjustments):
   - Remove project context (no project_summary in said-done-bot)
   - Keep: noise removal, ambiguity resolution, dependency ordering
   - Keep markers: `__NO_TASKS__`, `__TOO_MANY_TASKS__`, `__SUMMARY__`
   - Keep examples
   - Max 10 tasks limit

2. Create `core/task_extractor.py`:
   - `_get_prompt()` — lazy load from file
   - `_parse_response(response)` — parse numbered list or markers
   - `async extract_tasks(transcripts, llm_provider)` — join, truncate to 4000, call LLM, parse
   - `async llm_complete(system_prompt, user_message)` — wrap httpx call to OpenRouter

## Verification
```bash
python3 -c "
from core.task_extractor import _parse_response
r = _parse_response('1. Позвонить маме\n2. Купить хлеб')
assert r['tasks'] == ['Позвонить маме', 'Купить хлеб']
r2 = _parse_response('__NO_TASKS__')
assert r2['marker'] == 'no_tasks'
print('OK')
"
```
